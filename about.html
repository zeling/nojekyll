About this site

This site initially serves as a proof of concept that we don't need static site generators to write blogs, instead we can make the process really easy with Makefile and a good programming language. I plan to explain this in the near future and make it to be the first post of this blog.

I am not a web designer, so please bear with me for the terrible web design. I tried to make the content more readable, the CSS stylesheet of this site is based on (blatantly copied from) this site. The good news is that unlike some other sites, I won't add any suspicious particle effects whose meaning remains unclear.

Opinions expressed are solely my own and do not express the views or opinions of my employer.
